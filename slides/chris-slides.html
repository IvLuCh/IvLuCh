Modulo

  • Teilungsrest: $ 3 : 2 = 1\textrm{R}1 $
  • $ 3\ \textrm{mod}\ 2 = 1 $

Kongruenz

  • Ausführlich: $ x\ \textrm{mod}\ n = y\ \textrm{mod}\ n $
  • $ x \equiv y\ (\textrm{mod}\ n) $
  • $ 3 \equiv 5\ (\textrm{mod}\ 2) $

Summe

  • Ausführlich: $ f(k) + f(k + 1) + \ldots + f(m) $
  • 
						for(int n = k; n <= m; n++)
  • $ \sum\limits_{n=k}^{m} f(n) $
  • $ \sum\limits_{n=1}^{3} n = 1 + 2 + 3 = 6 $
  • $ \sum\limits_{n=1}^{\infty} 2^{-n} = \frac{1}{2} + \frac{1}{4} + \frac{1}{8} + \ldots = 1 $
